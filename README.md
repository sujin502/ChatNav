# 🧭 ChatNav - AI 对话沉浸式侧边栏

**让 AI 长对话更易管理！** 
ChatNav 为各大主流 AI 平台的网页版添加沉浸式侧边栏目录，支持自动生成大纲、一键回看历次提问和精美毛玻璃 UI，显著提升长文对话体验。

## ✨ 核心功能

*   **⚡ 自动生成目录**：实时扫描对话内容，自动提取用户提问生成侧边栏大纲。
*   **🖱️ 平滑跳转**：点击目录项即可平滑滚动到对应的对话位置。
*   **🎨 沉浸式 UI**：
    *   **毛玻璃特效**：现代化磨砂玻璃质感，适配深色/浅色模式。
    *   **可拖拽悬浮球**：不使用时自动收起为精美悬浮球，位置随意拖拽，支持记忆。
    *   **自定义配色**：默认粉白配色，清新优雅。
*   **🚀 多平台支持**：一个脚本，通吃主流 AI 模型。
*   **🔄 自动适配**：智能识别当前网站，自动加载对应的适配策略。

## 🛠️ 安装方法

### 方式一：Tampermonkey 油猴脚本 (推荐)

1.  安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)。
2.  [点击这里安装脚本](#) (如果有发布链接) 或创建一个新脚本，将 `ChatNav.js` 的内容复制进去。
3.  保存并刷新 AI 聊天页面即可生效。

## 📖 使用指南

1.  **打开侧边栏**：
    *   默认情况下，页面右侧会出现一个**粉色悬浮球**。
    *   鼠标**移动到悬浮球上**，侧边栏会自动展开。
2.  **浏览与跳转**：
    *   侧边栏列出了所有的提问记录。
    *   点击任意一条，页面会自动滚动到该问题的位置。
3.  **搜索**：
    *   在侧边栏顶部的搜索框输入关键词，可以快速过滤对话记录。
4.  **拖拽**：
    *   按住悬浮球或侧边栏顶部，可以将其拖拽到屏幕任意高度。
    *   位置会自动保存，刷新后依然有效。
